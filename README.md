# Project_Register_Customers

## Descrição

  Projeto que cadastra clientes no banco de dados, utilizando Java, Angular e PostgreSQL.

## O que foi utilizado no projeto?

- Java17
- Spring Boot
- Spring Tools Suite
- PostgreSQL
- PgAdmin4
